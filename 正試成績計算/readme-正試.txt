程式目的
	1.計算出 個人的 排名與PR值(縣市與整體)、成績含各向度、保留作答反應與二元作答反應
文件準備
	1. 需在縣市合併檔時加入 縣市的資訊欄位
	2. 需在全部合併後加入，缺考、補考 這兩項資訊欄位
	3. 需根據當時規則整理 原始作答反應的 例外選項如 多選
	4. 需根據該年度準備 各科目各年級的答案檔(須按照格式設定，參考程式碼 或 答案檔)


程式設定

	

	1. old.path: 原始資料夾路徑 (# choose.files())
	2. new.path: 資料夾內檔案清單( list.files(path = old.path,full.names = T)  )

	
狀況:

	1.讀入格式為 xlsx 的，檔案過大時讀取時間過久，整體一次執行約200分鐘
	2.讀入格式為 csv  的，讀取時間大幅減少，整體一次執行約80分鐘
	3.有部分縣市的 縣市PR值會有出入，目前暫定公式為 (考試人數-缺考-排名) / (考試人數-缺考)
	4.縣市PR值 若不在 excel 內 以 ( 縣市、年級、班級代碼、座號排序)，在資料回傳階段會出錯
